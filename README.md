# June 2026 Real-Time Daily Task Manager

A modern React + TypeScript task manager for June 2–30, 2026, with real-time task updates, analytics, calendar history, gamification, and export reports.

## Features

- Live current task tracking and countdown
- Daily & monthly productivity analytics
- Monthly calendar with status colors
- XP, levels, badges, streak tracking
- Browser notifications and optional sound alerts
- Local storage persistence for task state and history
- CSV and PDF export support
- Responsive dark UI with neon glassmorphism

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open the URL shown in the terminal.
